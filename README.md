# Install

You will need

- Node 11.8.0
- Yarn

```
yarn
```

# Run

1. Run the server
```
yarn start
```

2. Open the browser at `http://localhost:1234`

# Deploy to Dev
Run
```
yarn deploy-to-development
```
# Deploy to Staging
Run
```
yarn deploy-to-staging
```

# Deploy to Prod
Run
```
yarn deploy-to-production
```
